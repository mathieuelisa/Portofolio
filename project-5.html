ROOM est mon premier site realisé en autodidacte.

Dans un premier temps j’ai voulu m’exercer sur HTML, CSS ainsi que Javascript en reeditant à partir de zéro une page d’intégration permettant de s’entrainer.

Après avoir relevé ce défi j’ai rapidement souhaité approfondir mes connaissances en upgradant ce site en changeant sa page de présentation, en creant une page de contact ainsi qu’une page « about ».

J’ai également implementer Node.js au projet afin d’obtenir et gérer une database pour mon formulaire.

Les difficultes de ce projet ont ete le fait d'integrer certaines librairies.

Frameworks & Libs

  • JAVASCRIPT
  • NODE JS
  • EXPRESS JS
